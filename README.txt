הפרויקט נמצא בכתובת הזאת
 http://beconnectedd.000webhostapp.com/

 **  בשלב ראשון של הפרויקט האפליקציה היא ברמת FrontEnd **

האפליקציה משמשת כרשת חברתית ושיתוף מידע בין חברים.
ניתן להירשם ולשתף תמונות וקטעי טקסט קצרים בכדי שמשתמשים אחרים יוכלו לראות ולהגיב.
ניתן לשלוח הודעות בין משתמשים מחוברים.


הפרויקט נכתב בסביבת עבודה VS code עם Package node.js   ו- Angular CLI



בכדי להריץ את הפרוקיט :

-נבצע clone או downloadZip לתיקייה חדשה במחשב.
-נפתח את ה VS CODE ונבחר את התיקייה של הפרויקט.
-נריץ את הפקודה npm install ב- Terminal של ה Vs code
-לאחר מכן נריץ את הפקודה npm install -g @angular/cli
-לאחר מכן נריץ את הפקודה ng serve 
- כעת הפרויקט יופיע באתר בכתובת http://localhost:4200/
